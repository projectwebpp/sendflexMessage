# Plan 01-01: Backend Foundation

## Objective
Bootstrap the Express.js project with folder structure, Supabase config, response utilities, error middleware, and app/server entry points. Delivers a running server skeleton — CRUD routes are wired in plan 01-02.

---

## Database Schema (reference for plan 01-02)

Table: `tasks` (Supabase / PostgreSQL)

| Field | Type | Notes |
|-------|------|-------|
| id | uuid | Primary key, default gen_random_uuid() |
| title | text | NOT NULL |
| description | text | nullable |
| status | boolean | default false |
| created_at | timestamptz | default now() |

<task type="auto" id="03">
  <name>Error Middleware</name>
  <files>
    - backend/src/middlewares/errorMiddleware.js
  </files>
  <action>
    Write backend/src/middlewares/errorMiddleware.js:

    exports.notFound = (req, res, next) => {
      res.status(404).json({ success: false, data: null, message: `Route not found: ${req.originalUrl}` })
    }

    exports.errorHandler = (err, req, res, next) => {
      console.error(err.stack)
      const statusCode = res.statusCode !== 200 ? res.statusCode : 500
      res.status(statusCode).json({ success: false, data: null, message: err.message || 'Internal server error' })
    }

    Note: errorHandler MUST have exactly 4 parameters for Express to treat it as an error-handling middleware.
  </action>
  <verify>
    <automated>node -e "const m=require('./backend/src/middlewares/errorMiddleware.js'); if(typeof m.notFound!=='function') throw new Error('notFound missing'); if(m.notFound.length!==3) throw new Error('notFound must have 3 params'); if(typeof m.errorHandler!=='function') throw new Error('errorHandler missing'); if(m.errorHandler.length!==4) throw new Error('errorHandler must have 4 params (err,req,res,next)'); console.log('PASS')"</automated>
  </verify>
  <done>
    - middlewares/errorMiddleware.js exports notFound (arity 3) and errorHandler (arity 4)
    - notFound sends { success: false, message: "Route not found: ..." } with status 404
    - errorHandler sends { success: false, message } with status 500 (or res.statusCode if already set)
  </done>
</task>

<task type="auto" id="04">
  <name>App and Server Bootstrap</name>
  <files>
    - backend/src/app.js
    - backend/src/server.js
  </files>
  <action>
    1. Write backend/src/app.js:
       - require('dotenv/config')
       - const express = require('express')
       - const cors = require('cors')
       - const { notFound, errorHandler } = require('./middlewares/errorMiddleware')
       - const app = express()
       - app.use(cors())
       - app.use(express.json())
       - // Task routes will be mounted here in plan 01-02:
       - // app.use('/tasks', require('./routes/taskRoutes'))
       - app.use(notFound)
       - app.use(errorHandler)
       - module.exports = app

    2. Write backend/src/server.js:
       - const app = require('./app')
       - const PORT = process.env.PORT || 3001
       - app.listen(PORT, () => console.log(`Server running on port ${PORT}`))
  </action>
