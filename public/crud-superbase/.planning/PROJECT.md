# PROJECT.md

## Project Name
Full Stack CRUD Task Management System

## Description
Enterprise SaaS Dashboard for task management with production-ready UX/UI. Full-stack application with React frontend, Express.js backend, and Supabase database.

## Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- SweetAlert2
- Axios
- React Router DOM
- Zustand

### Backend
- Node.js
- Express.js
- dotenv
- cors

### Database & Auth
- Supabase (PostgreSQL)

## Deployment
- Frontend: Vercel
- Backend: Render

## Constraints
- All API responses must follow a standard response envelope
- ENV variables must never be exposed in source code
- UI must support dark and light themes with persistent state
- Must be responsive across mobile, tablet, laptop, desktop, ultra-wide
- Glassmorphism design language — reference: Linear, Notion, Vercel Dashboard, Stripe Dashboard
- Inter font only
