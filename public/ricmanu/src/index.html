จัดการ Rich Menu

ดูและจัดการ Rich Menu ที่มีอยู่ทั้งหมด

ต้องตั้งค่าครั้งแรกก่อนใช้งาน

ระบบ Expiry ต้องการ Trigger เพื่อทำงานอัตโนมัติ ทำตามขั้นตอนนี้ 1 ครั้ง แล้วกด "สร้าง Trigger" ด้านล่าง

  1. เปิด Google Sheet → Extensions → Apps Script
  2. dropdown function → เลือก initializePermissions
  3. คลิก ▶ Run → Google ขอสิทธิ์ → คลิก Allow
  4. กลับมากดปุ่ม ตรวจสอบสิทธิ์ ด้านล่างเพื่อยืนยัน แล้วกด สร้าง Trigger
